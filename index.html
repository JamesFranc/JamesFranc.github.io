I enjoy making tools that people like using, but
tinkering to see if I can make something work is a passion, of sorts.

A few years back after switching from chemistry to computer
science, I began programming and haven't looked back. This page
is a return to the basics project for me using only HTML, CSS and JS.

My goal for the project was to solidify knowledge
of these foundational elements for web based software.
Designing an attractive site using the basics took some time, and
a few iterations to get where it is. Responsiveness and mobile
compatibility were key objectives in working on the project.
Along the way I got to learn about blocking JS and CSS,
server-side configuration, browser caching, image optimization,
Google's PageInsights and how mobile and desktop pages score.

In previous projects I've used AWS and their myriad of
wonderful services. For exposure and learning's sake I chose
to use Digital Ocean, and have been very pleased with it's ease
of use and functionality.

My aim is to create a responsive, clean website that
could be used to display my technical abilities and any
knowledge gained in the process of making.
